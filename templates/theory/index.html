1. Основные понятия

• Внутренняя энергия (U) — энергия движения и взаимодействия частиц.

Для идеального одноатомного газа:

U=(3/2)νRT (где ν — количество вещества, R — газовая постоянная, T — температура)

• Работа газа (A) — определяется площадью под графиком в координатах P−VP−V:

A = PΔV (для изобарного процесса)

• Первое начало термодинамики:

Q = ΔU+A (Q — теплота, ΔU — изменение внутренней энергии)

• Если газ получает тепло:

Q>0Q>0;

Отдаёт:

Q<0Q<0.

• Работа газа положительна при расширении (A>0) и отрицательна при сжатии (A<0).

2. Изопроцессы

Процесс Условие Уравнение Внутренняя энергия (ΔU) Работа (A) Теплота (Q)
Изотермический T=const PV=const ΔU=0 A=νRTlnV2/V1 Q=A
Изобарный P=const V/T=const ΔU=(3/2)νRΔT A=PΔV Q=(5/2)νRΔT
Изохорный V=const P/T=const ΔU=(3/2)νRΔT A=0 Q=ΔU
Адиабатный Q=0 PVγ=const(γ=Cp/Cv) ΔU=−A A=−3/2νRΔT Q=0

3. Тепловые машины и циклы

• КПД тепловой машины:

η = A/Q1 = 1−Q2/Q1 (Q1 — тепло от нагревателя, Q2 — тепло холодильника)

• Цикл Карно (максимальный КПД):

η = 1−T2/T1 (T1 — температура нагревателя, T2 — холодильника)

• Второе начало термодинамики:

Невозможно создать вечный двигатель второго рода (всё тепло нельзя превратить в работу).

4. Теплоёмкость

• Теплоёмкость при постоянном объёме (Cv):

Cv = 3/2R (для одноатомного газа)

• Теплоёмкость при постоянном давлении (Cp):

Cp = 5/2R (для одноатомного газа)

5. Графики процессов

• P−V диаграмма:

- Площадь под кривой = работа газа.

- Изотерма — гипербола, изобара — горизонтальная линия, изохора — вертикальная линия.

• T−V диаграмма:

- Изобара — прямая, проходящая через начало координат.

6. Типичные задачи

1. Расчёт работы газа в цикле:

• Найти работу на каждом участке и сложить.

2. Определение КПД тепловой машины:

• Использовать формулу η = 1−Q2/Q1.

3. Применение первого начала термодинамики:

• Учесть знаки Q, ΔU и A.

7. Единицы измерения

• Давление (P): Па (Паскаль).

• Объём (V): м³.

• Температура (T): К (Кельвин).

• Газовая постоянная (R).

Важно!

• Не путать знаки теплоты и работы.

• Для адиабатного процесса Q = 0, поэтому ΔU = −A.

• В задачах с графиками всегда анализируйте тип процесса по форме кривой.

Пример задачи:

Газ совершил цикл, состоящий из изотермического расширения (A = 200Дж) и изохорного охлаждения. Определите количество теплоты, полученное газом за цикл.

Решение:

В изотермическом процессе Q = A = 200Дж. В изохорном процессе A = 0, Q = ΔU. Так как цикл замкнут, суммарное ΔU = 0, а Q цикла = A цикла=200Дж. {% endblock %}
